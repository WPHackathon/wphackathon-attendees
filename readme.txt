Creates a environment to manage the Attendees participating in the WPHackathon

== Description ==

This plugin creates:
* A brand new custom post type to manage attendees in your backend.
* A form page to allow attendees to sign up on your page. This registrations should be approved afterwards.
* A cute attendees list page.

All the attendees are segmented by their main skill, and could have some data like twitter or email.

This plugin was created for [WPHackathon](https://www.wphackathon.com) and had a little boost during [WPESCamp Bilbao 2017](https://wpes.camp/).


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wphackathon-attendees` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

Note that this plugins will create necessary pages and default skills for the members (you could change this skills afterwards).

== Frequently Asked Questions ==

= Do I have to keep the default skills? =

No, you don't. You could change / delete / create new ones.
